Here are a few good points worth mentioning about C.

C is........

  • Structured
    1. Has for and while loops
    2. Block structures (like when you "block" your code in curly braces)
    3. Subroutines (things that do small tasks like functions or methods)

  • Imperative (or procedural)
    1. Describes the program in terms of statements
    2. Opposite of "Declarative". Doesn't tell the program what to do, but how to do it

  • Useful for system programming
    1. Low level access to memory
    2. Communicates directly with machine language (Assembly) and hardware
