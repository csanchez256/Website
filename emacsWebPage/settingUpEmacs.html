1. Copy and paste the text (in red) below in your .emacs file
      (note: you can access .emacs by using keyboard macros C-x C-f then type ~/.emacs in mini-buffer)

      ;;; Always do syntax highlighting
      (global-font-lock-mode 1)

      ;;; Also highlight parens
      (setq show-paren-delay 0 show-paren-style 'parenthesis) (show-paren-mode 1)

      ;;; This is the binary name of my scheme implementation
      (setq scheme-program-name "scm")

    2. On the last line change "scm" to "mzscheme"

    3. Save file

    4. Download Racket
      sudo apt-get install racket

    5. Create your first scheme program and save it

    6. Run Scheme process (this is done in emacs text editor)
      M-x run-scheme

    7. You are now using racket interpreter, load your scheme program by typing:
      (load "your_program.scm")

    8. Now you're ready to go!
